# Changelog

All notable changes to Content Gravity SEO Toolkit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-31

### Added
- **Content Analyzer** with real-time readability scoring (Flesch Reading Ease)
- **Table of Contents (TOC)** generator with multiple styles (Boxed, Minimal)
- **Auto-insert TOC** functionality with position options
- **Shortcode** `[cg_toc]` support for manual placement
- **Internal Link Assistant** with smart link suggestions
- **Link Plan Builder** with one-click link insertion
- **Rollback functionality** for bulk link operations (24-hour window)
- **Link Juice Calculator** using PageRank-inspired algorithm
- **Orphan Page Finder** to identify pages with no inbound links
- **Weak Page Detection** for low-authority content
- **Content Clusters** visualization with hub detection
- **HTTP Status Checker** for broken link scanning
- **Focus keyword tracking** with density analysis
- **Heading structure validation**
- **Image-to-text ratio checking**
- **Dark mode support** across admin interface
- **Elementor compatibility** for link extraction
- **Collapsible/expandable TOC** option
