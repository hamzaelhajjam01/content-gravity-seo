# Content Gravity SEO

Content Gravity SEO is a premium-grade WordPress SEO toolkit focused on **content optimization** and **internal link authority management**.

The plugin is designed for site owners who want to improve SEO structure, not rely on black-box automation.

---

## Key Features

### Content Analyzer
- Live content analysis in Classic Editor and Gutenberg
- Optimization score (A–F)
- Word count, heading structure, readability checks
- Focus keyword tracking
- Internal and external link analysis

### Internal Linking Assistant
- Internal link graph indexing
- Link Juice (internal authority) score
- Orphan page detection
- Weak page detection
- Manual, safe internal link insertion

### Table of Contents (TOC)
- Automatic TOC generation based on headings
- Gutenberg TOC block
- Boxed or in-content display
- Optional collapse toggle
- Built for stability and theme compatibility

### Tools & Maintenance
- Rebuild internal link index
- Recalculate authority scores
- Import / Export settings

---

## Editor Compatibility

- ✅ Classic Editor
- ✅ Gutenberg (Block Editor)
- ❌ Elementor auto-insertion (detected and safely disabled)

Elementor content is analyzed, but automatic content modification is disabled to prevent layout issues.
